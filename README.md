# Appointment Booking System

## Overview
Full-stack appointment booking system with role-based authentication.

- **Frontend:** HTML, CSS, JS
- **Backend:** Node.js, Express, JSON files as DB
- **Features:**
  - User login & booking
  - Admin login & view all bookings
  - Role-based page protection
  - Logout

## How to run
1. `cd backend && npm install && node server.js`
2. Open `frontend/index.html` in browser
3. Login as user/admin

## Notes
- All data stored in JSON files
- Node.js backend required
